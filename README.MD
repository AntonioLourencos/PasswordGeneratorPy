# PASSWORD GENERATOR

## Minium requirements

Python3

## Installing

```sh

    python3 requirements.py

```

## Running

```sh

    # Arg is optional

    # arg1
    # type: string
    # options: one or multiple

    # arg2
    # type: integer
    # options: >=8

    # Default arg
    # arg1 = multiple
    # arg2 = 8

    python3 lib arg1? arg2? 

```

### Running Extra

```sh

    python3 lib customParams

```
